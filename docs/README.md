# Sreda Docs

Эта папка содержит документацию по `Среде`.

Структура:
- [public](./public/README.md) - очищенный набор документов, который можно готовить к публикации в публичный репозиторий;
- [internal](./internal/README.md) - внутренние документы, operational notes и непубличные решения.

Принцип:
- в корне `docs` не держим дублирующиеся markdown-файлы;
- public и internal живут отдельно;
- корневой `README.md` служит только навигацией.
