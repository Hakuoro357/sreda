# Sreda Docs

Эта папка содержит документацию по `Среде`.

Структура:
- [public](./public/README.md) - набор документов, который можно публиковать в открытый репозиторий.

Принцип:
- в корне `docs` не держим дублирующиеся markdown-файлы;
- в публичном репозитории хранится только publishable documentation;
- корневой `README.md` служит только навигацией.
